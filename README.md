## 🌐 Project Overview
Developed a premium, single-page professional website for the **Institute of Digital Risk (IDR)**. The design focuses on a "high-consequence" aesthetic, bridging the gap between academic depth and industrial resilience.

## ✨ Key Features

### 1. **Premium 3D Visuals**
- **Isometric Assembly:** A custom-engineered, animated 3D visual in the hero section representing complex risk structures.
- **Glassmorphism:** Sublte blur effects and layered geometries provide a sophisticated, modern UI.

### 2. **Innovative Hybrid Design**
- **Contrast Theming:** High-contrast sectional themes (Light Home/About & Dark Service/Contact/Footer) to maintain engagement and professional hierarchy.
- **Micro-Animations:** Smooth scroll-reveal effects using Intersection Observer API.

### 3. **Mobile-First Responsiveness**
- **Full-Screen Nav Cabinet:** A custom full-screen navigation container with standard close (X) controls for a native-app feel on mobile.
- **Responsive Grids:** All service pillars, feature cards, and contact forms stack dynamically for all device breakpoints (Phone, Tablet, Desktop).

### 4. **Modern UI/UX**
- **Premium Typography:** Utilizes **Outfit** and **Space Grotesk** for a tech-focused, readable experience.
- **Dark Contact Interface:** A sleek, innovative dark card design for the contact section with interactive glowing focal points.

## 🛠️ Tech Stack
- **HTML5:** Semantic architecture for SEO and accessibility.
- **CSS3:** Custom properties (CSS Variables), Grid, Flexbox, and Hardware-accelerated animations.
- **JavaScript (Vanilla):** Light-weight logic for navigation, scroll interactions, and form simulations.
- **Version Control:** Git & GitHub integration.
- **Deployment:** Vercel for high-performance global hosting.
